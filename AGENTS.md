# AGENTS.md

## 原則

- このプロジェクトで作成する成果物は、原則として日本語で作成する
- UI文言、生成される予言文、バリデーションメッセージ、AI用プロンプト、ユーザー向けドキュメントは日本語にする
- コード上の識別子、型名、関数名、変数名、ファイル名、設定キー、ライブラリ由来の値、テスト名などは、TypeScript/React/Viteの慣習に従って英語でよい
- 既存コードや外部API、ライブラリの仕様で英語が自然な箇所は無理に日本語化しない

## プロダクトの目的

「ラブリーゴーストライター」をモチーフにした、日本語のみの非公式ファンWebアプリを作る。ユーザーが軽い個人情報や相談テーマを入力すると、1ヶ月分 / 4週分 / 各週1行の象徴的な予言を生成し、任意の生成AIへ貼り付けるためのコピー可能なプロンプトも生成する。

## 使用技術

フロントエンド単体SPA

- React
- Vite
- TypeScript
- Tailwind CSS
- MUI
- Vitest

## 機能要件

`docs/` 配下を参照する

## ドキュメント運用

- 実装判断が現在の設計から意図的に外れる場合は、関連ドキュメントを更新する
- 非公式ファン作品の注意書きや安全制約を削除・弱体化しない
- 生成例やテンプレートは、現実の危害や断定的予言を避ける方針と一致させる
