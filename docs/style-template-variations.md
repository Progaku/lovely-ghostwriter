# 文体・テンプレート・AIプロンプト対応表

## 1. 目的

このドキュメントは、ラブリーゴーストライター風 Web アプリで使う「文体」と「テンプレート」の候補を整理し、それぞれを生成 AI 用プロンプトへ対応付けるための設計資料である。

本アプリは非公式ファン作品であり、生成結果は娯楽用途の文章として扱う。文体は象徴的、不穏、暗示的な雰囲気を持たせるが、現実の危害、病気、死、犯罪、金銭損失などを断定的に予告しない。

## 2. 共通方針

### 2.1 予言文の共通制約

- 日本語で出力する。
- 1ヶ月分を4週に分ける。
- 各週は必ず4行にする。
- 入力値のうち、名前、相談テーマ、今月の気分を自然に反映する。
- 生年月日と性別は、必要な場合のみ象徴的に反映する。
- 直接的な不幸の予告ではなく、解釈の余地がある暗示に留める。
- 最終行は、読後に小さな行動や見直しへつながる余地を残す。

### 2.2 禁止または回避する表現

- 「必ず失敗する」「不幸になる」などの断定。
- 死、病気、事故、犯罪、破産などを直接予告する表現。
- 医療、法律、金融の専門判断を代替する助言。
- ユーザーを怖がらせるためだけの煽り。
- 公式素材、公式画像、既存キャラクター名への依存。

### 2.3 差し込み変数

テンプレートでは以下の変数を使う。

```text
{name}       ユーザー名
{theme}      相談テーマ
{mood}       今月の気分
{week}       週番号
{symbol}     象徴語
{place}      場所語
{object}     道具語
{action}     穏当な行動語
{toneHint}   文体ごとのニュアンス
```

## 7. AIプロンプト雛形

```text
あなたは日本語で、占い風の四行詩を現実的な助言に読み替えるアシスタントです。
以下のユーザー入力と四行詩を読み取り、相談テーマに関係する助言を返してください。

名前: {name}
生年月日: {birthDate}
性別: {gender}
相談テーマ: {theme}
今月の気分: {mood}

【文体】
{styleName}

【四行詩】
{weeks}

【出力条件】
- 日本語で出力する
- 四行詩の内容を助言の根拠として扱う
- ユーザーの相談テーマに直接関係する助言を返す
- 不穏さを煽らず、次に取れる小さな行動へ落とし込む

{interpretationAxis}

1. 四行詩から読める今月の流れ
```

## 8. 実装時の選択ロジック案

### 8.1 文体選択

入力値由来のシードと再生成用の `randomSalt` を組み合わせ、文体を1つ選ぶ。相談テーマに含まれる語から候補を絞ってもよい。

| 入力傾向 | 優先文体 |
| --- | --- |
| 恋愛、連絡、相手、友人 | S04, S05, S11, S22, S28 |
| 仕事、勉強、締切、転職 | S03, S06, S10, S19, S31 |
| 不安、疲れ、迷い | S08, S17, S18, S21, S26 |
| 創作、習慣、生活 | S07, S14, S24, S29, S30 |
| 将来、進路、選択 | S06, S09, S15, S23, S25 |

### 8.2 テンプレート選択

文体 ID に紐づくテンプレート候補から1つ選ぶ。週ごとにテンプレートを変える場合でも、1ヶ月全体で文体の統一感を保つ。

```text
styleId -> candidateTemplateIds -> seededRandom selection
```

## 10. 実装メモ

- 文体 ID、テンプレート ID、解釈軸は定数化する。
- 禁止語彙は生成結果に含まれないよう、テンプレート側で避ける。
- AI用プロンプトには、実際に選ばれた文体名と解釈軸を含めると、助言の方向が安定する。
- 4週分すべてを同じ文体で統一し、週ごとにテンプレートまたは語彙だけを変えると読みやすい。
- 再生成では `randomSalt` により文体や語彙を変えつつ、相談テーマから大きく外れないようにする。
